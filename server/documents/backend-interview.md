## 1. Tư duy lập trình & Cấu trúc dữ liệu
- Big O
- Cấu trúc dữ liệu: HashMap, Array, Linked List, Tree
- Giải thuật: Sorting, Searching, hoặc Dynamic Programming

## 2. Kiến thức chuyên sâu về ngôn ngữ, framework
- Cơ chế hoạt động
- Memory management
- Concurrency & Multithreading

## 3. Database
- SQL vs NoSql: Khi nào dùng cái nào?
- Query optimization: Các đánh Index sao cho đúng
- Transactions: ACID là gì? Cách xử lý khi có hàng ngàn Transactions cùng lúc
- Database Design: Thiết kế Schema, chuẩn hoá (Normalization) và phản chuẩn hoá và giải thích

## 4. System Design:
- Scalability: Làm sao để hệ thống chịu tải từ 1k - 1tr users (Load Balancer, Horizontal scaling)
- Caching: Redis / Memcached để giảm tải cho Database
- Microservices: Cách các dịch vụ giảo tiếp với nhau (REST, gRPC, Message Queue như RabitMQ / Kafka)

## 5. Bảo mật & API
- Authentication / Authorization: Phân biệt JWT, OAuth2, SEssion / Cookie
- Security standard: Các lỗi phổ biến như SQL Injection, XSS, CSRF
- Thiết kế RESTful API chuẩn chỉnh, dễ mở rộng

## 6. Kĩ năng thực chiến (Devops & Tools)
- Docker & Kubernetes: Đóng gói và triển khai ứng dụng
- CI/CD: Hiểu quy trình tự động hoá việc test và deploy
- Git: Cách quản lí source code (Gitflow, Rebase vs Merge)
